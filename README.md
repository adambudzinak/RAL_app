**RAL App (SK)**
=============================
Výpočet periódy pre zadanú postupnosť mod p (prístup cez zadanie charakteristického polynómu, alebo aj postupnosti dĺžky n)

*Požiadavky*
----------------
* **Knižnica NTL**: Vyžaduje sa pre výpočty
